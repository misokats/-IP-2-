#課題：IPアドレスを「2進数」に変換せよ！

【問題】
ユーザーから入力された「10進数のIPアドレス（例：192.168.10.1）」を、**「各ドットごとに8桁の2進数」**に変換して表示するプログラムを作成してください。

入力例: 192.168.10.1

期待される出力例: 11000000.10101000.00001010.00000001


【ヒント：実装のステップ】
文字列を分割する:
Pythonの .split('.') を使うと、ドットで区切られた4つの数字をリストに分けられます。

数値に変換する:
リスト内の各要素（文字列）を int() で数値にします。

2進数に変換する:
組み込み関数の bin() を使います。

注意点: bin(192) は '0b11000000' という文字列を返します。頭の '0b' は不要なので工夫が必要です。

「8桁」に揃える（ここが重要！）:
例えば 1 を2進数にすると 1 だけになりますが、IPアドレスの世界では 00000001 と8桁で書くのがルールです。

.zfill(8) というメソッドを使うと、足りない桁を0で埋めてくれます。

結合する:
最後にまたドットで繋いで表示します。
